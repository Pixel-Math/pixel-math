Capítulos

Etapas do Projeto

1. Familiarização com os Arquivos LaTeX

  • Estudo da estrutura do livro em LaTeX (pastas, comandos, pacotes utilizados)
  • Identificação de elementos especiais (figuras, equações, referências internas)
  • Compreensão do fluxo de compilação LaTeX

2. Conversão para XHTML com LaTeXML

  • Instalação e configuração do LaTeXML
  • Conversão dos arquivos .tex para .xhtml
  • Verificação e ajustes no código gerado, quando necessário
  • Tratamento de equações e elementos matemáticos

3. Criação do ePub com Calibre

  • Organização dos arquivos XHTML, imagens e metadados
  • Utilização do Calibre para compilar o conteúdo em um arquivo .epub
  • Implementação de navegação entre capítulos
  • Testes de visualização em diferentes leitores de ePub

4. Criação da Página Web

  • Aproveitamento dos arquivos XHTML para montar a página web navegável
  • Adição de estilos CSS para melhorar a experiência de leitura
  • Implementação de recursos de navegação por capítulos
  • Otimização para diferentes dispositivos e tamanhos de tela
